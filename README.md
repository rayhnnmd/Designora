## Features

### Core Editor

* Add & edit **text, shapes, and images**
* Drag, resize, rotate elements
* Layer system with reorder & lock

### Smart Design Tools

* Smart alignment guides (like Figma)
* Undo / Redo system
* Duplicate elements (Ctrl + D)

### Text Styling

* Font family & size controls
* Bold (Ctrl + B), Italic (Ctrl + I)
* Shadow & outline effects

### Canvas Controls

* Custom canvas sizes
* Instagram / A4 presets
* Zoom in/out system

### Export

* PNG / JPG export
* High quality option
* Transparent background support

### Authentication

* Google Sign-in (Firebase)

---

## Tech Stack

* **Frontend:** Next.js (React)
* **Canvas Engine:** Fabric.js
* **State Management:** Zustand
* **Auth:** Firebase
* **Deployment:** Vercel

## Installation

```bash
git clone https://github.com/your-username/designora.git
cd designora/client
npm install
npm run dev
```
